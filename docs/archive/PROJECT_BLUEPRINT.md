# AIC Website Development Project Plan

## Project Overview
**Project Name**: AI Integrity Certification (AIC) Website
**Objective**: Build a lead-generation engine for the Alpha Program.

## Strategic Objectives
1.  **Lead Generation**: Convert 10-15 enterprise prospects.
2.  **Thought Leadership**: Establish AIC as the POPIA Section 71 authority.
3.  **Trust Building**: Professional design (Crimson Pro / Editorial).

## Site Architecture
1.  **Homepage**: Value prop + Alpha CTA.
2.  **About**: Origin story + "Trust Gap" narrative.
3.  **Tier Framework**: Deep dive into the 3-Tier model.
4.  **Certification Process**: Timeline and pricing transparency.
5.  **Self-Assessment**: Interactive quiz (Lead Magnet).
6.  **Alpha Program**: Application form.

## Design Guidelines
*   **Typography**: Crimson Pro (Headings), IBM Plex Mono (Body).
*   **Colors**: Red (Tier 1), Orange (Tier 2), Green (Tier 3), Gold (Accent).
*   **Vibe**: Editorial newspaper meets modern tech.
