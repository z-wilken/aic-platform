# AIC Documentation

> **AI Integrity Certification** — The first POPIA Section 71 compliant accountability framework for South African AI.

## Core Principle

> "We do not regulate AI. We certify that humans remain accountable."

## Three-Tier Framework

| Tier | Name | Human Role | Example |
|------|------|------------|---------|
| **1** | Human-Approved | AI advises, human decides | Cancer treatment, parole |
| **2** | Human-Supervised | AI executes, human oversees | Loan approvals, hiring |
| **3** | Automated-Permissible | AI operates, periodic check | Recommendations, filters |

See **[tier-framework.html](design/tier-framework.html)** for visual representation.
