## Three-Tier Framework

| Tier | Name | Human Role | Examples |
|------|------|------------|----------|
| **1** | Human-Approved | AI advises, human decides | Cancer treatment, parole, credit denial appeals |
| **2** | Human-Supervised | AI executes, human oversees | Loan approvals, resume screening, fraud flags |
| **3** | Automated-Permissible | AI operates, periodic review | Recommendations, email filters, chatbots |

See [tier-framework.html](design/tier-framework.html) for visual representation.
